# TTT-FOT-Giver-And-Lootable

TTT Lootable Props and Floor Weapon Giver addons, adapted for use with the Garry's Mod TTT rehaul Fistful of Traitors.

Requires Fistful of Traitors to work, which can be found here: https://steamcommunity.com/workshop/filedetails/?id=2881307920

Original Lootable Props addon can be found here: https://steamcommunity.com/sharedfiles/filedetails/?id=654149429

Original Floor Weapon Giver addon can be found here: https://steamcommunity.com/sharedfiles/filedetails/?id=2592321966

Convars "ttt_floor_weapons_giver_ammo" and "ttt_lootable_weapon_drop_chance" have been removed due to the weapons not needing to pick up any ammo, thus making the original convars useless.
